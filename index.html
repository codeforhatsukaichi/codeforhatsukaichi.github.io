Code for Hatsukaichi

About Us

Code for Hatsukaichiは、デザインとICT(情報通信技術)の力を用いて、廿日市のまちづくりを推進するために活動する団体です。

行動指針

  1. 廿日市のまちづくりに関心があれば誰でも活動に参加できます。
  2. 互いの自主性を尊重しながら活動に参加します。
  3. 互いの自律性を尊重し対等な関係で活動に参加します。
  4. 互いの特性を大切にして活動に参加します。
  5. 互いの情報を共有しながら活動に参加します。
  6. 互いの信頼関係に基づいて活動に参加します。
  7. 次代につながる人づくりをしながら活動に参加します。
